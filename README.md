- Title: Jeju Credit Card Regression
- Goal: Estimate the total amount of monthly credit card usage by region and industry
- Category: Regression
- Tool Used: python, pandas, sklearn
- Period: June, 2020 ~ July, 2020

# Data
- Data: Credit Card Data
- Shape: (26,048,114 * 12)
